1. Query - 기본 select

   select 컴럼명1, 컬럼2.....

   from 테이블

   * sql문은 대소문자 구분하지 않는다.

   - ;은 sql문의 종료를 의미

   - ;을 입력하기 전에 여러 줄로 sql문을 작성할 수 있다.

   - *은 모든 컬럼을 조회하겠다는 의미

   - 컬럼에 null을 저장할 수 있다.

     null은 0이나 space만 입력해 놓은 값과 다른 값

     null은 아무값도 없는 것을 의미

     사용할 수 없고 자장되어 있지 않은 상태
